Portfolio

Every since I was a kid, I always loved creating things that work. I had an unusual fascination with kinetic structures like marble tracks, water fountains, and gear-based structures. This evolved into a love of LEGOs, specifically their Technic line. This love for building combined with my love of video games led me to Minecraft and its automation-based gameplay. The game has blocks called Command Blocks, which function as an in-game coding interface. This is where my love for programming started. Since then, I have moved away from the game to focus on programming in actual programming languages. The languages I know fluently are:

  • Python 2 & 3
  • Java
  • HTML
  • CSS
  • Javascript
  • C
